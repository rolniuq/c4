## Goal: Build a Todolist App

### Description
Create a simple todolist web application that allows users to add, complete, and delete tasks.

### Requirements
- Add new tasks with a text input
- Mark tasks as complete (toggle)
- Delete tasks
- Tasks persist in localStorage
- Clean, simple UI

### Tech Stack
- Plain HTML/CSS/JavaScript (no framework)
- Single page application

### Acceptance Criteria
- [ ] User can add a new task
- [ ] User can click to mark task as complete
- [ ] User can delete a task
- [ ] Tasks remain after page refresh
- [ ] UI is clean and usable
